# TempoKid — 節拍遊戲

幫助小朋友練習節拍穩定度的小遊戲。純前端單一 HTML，手機/平板瀏覽器直接開就能玩。

## 模式

### 🥁 打拍子挑戰

1. 選速度（預設 88 BPM，滑桿是傳統節拍器刻度，附速度術語）
2. 按「開始」，先聽 4 拍抓感覺
3. 跟著節拍點大按鈕，彈出 PERFECT / GREAT / GOOD / MISS 判定與連擊
4. 偏掉時會提示「搶拍了，慢一點」或「慢半拍了，快一點」
5. 準確率 80% 以上就試試加快速度

### 🙈 節拍器躲貓貓

節拍器出聲 4 拍後躲起來變靜音，要靠心裡的拍子繼續點。
點偏太多（MISS）或停下來就會「被抓到」，看能撐幾拍，最佳紀錄存在 localStorage。
這是把內在拍感練起來最對症的練習。

### 🐤 節奏小勇士（Patapon 式）

四顆鼓：啪（中鼓）、碰（清音）、恰（高鈸）、咚（大鼓）。
聽 4 拍鼓語口令，照節拍打回去，帶小雞前進衝終點：

- 啪啪啪碰 ＝ 前進
- 碰碰啪碰 ＝ 攻擊（👾 怪物）
- 恰恰啪碰 ＝ 防禦（🪨 落石）
- 咚咚恰恰 ＝ 衝鋒（👹 大魔王）

有 3 顆心，口令失敗扣 1 顆並退一格，扣完遊戲結束。
口令連續成功 3 次進入 FEVER，分數 ×2。電腦鍵盤：D＝啪、F＝碰、J＝恰、K＝咚。

## 開發

沒有 build step，直接用瀏覽器開 `index.html` 即可。

```powershell
start index.html
```
