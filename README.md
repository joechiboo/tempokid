# TempoKid — 打拍子挑戰

幫助小朋友練習節拍穩定度的小遊戲。純前端單一 HTML，手機/平板瀏覽器直接開就能玩。

## 玩法

1. 選速度（建議從 80 BPM 開始）
2. 按「開始」，先聽 4 拍抓感覺
3. 跟著節拍點大按鈕，即時看準不準
4. 準確率 80% 以上就試試加快速度

## 開發

沒有 build step，直接用瀏覽器開 `index.html` 即可。

```
start index.html
```
